Описание проекта

Веб приложение по регистрации нарядов созданное с помощью фреймворка .Net Core.

Данное приложение разработано с целью оптимизации рабочего процесса при работе с нарядами на подключение и отключения электроустановок потребителей.

Функционал:

  • Регистрация пользователей.
  • Разграничение прав пользователей на основе ролей.
  • Разграничение доступа к информации в разрезе филиалов и участков.
  • Сохранение нарядов в pdf файле для последющей печати
  • Отслеживание сроков выполнения нарядов
  • СМС оповещение пользователей об изменении статуса наряда (в процессе разработки)
